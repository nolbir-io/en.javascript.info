# Xususiyatlarni jamlang

O'zboshimchalik bilan ish haqi miqdoriga ega `salaries` ob'ekti mavjud.

`Object.values` va `for..of` siklidan foydalanib, barcha ish haqilar yigâ€˜indisini qaytaruvchi `sumSalaries(salaries)` funksiyasini yozing.

Agar `salaries` bo'sh bo'lsa, natija `0` bo'lishi kerak.

Masalan:

```js
let salaries = {
  John: 100,
  Pete: 300,
  Mary: 250,
};

alert(sumSalaries(salaries)); // 650
```
